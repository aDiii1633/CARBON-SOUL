# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT report it via public GitHub issues.

Instead, please send an email to the repository maintainer directly, or use GitHub's private vulnerability reporting feature if enabled on the repository. We will acknowledge your report promptly and outline the next steps for resolving the issue safely.
