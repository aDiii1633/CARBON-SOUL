# Contributing to Carbon Soul

Thank you for investing your time in contributing to Carbon Soul!

## How to submit a PR
1. Fork the repo and create a feature branch.
2. Run `npm install` to install dependencies.
3. Make your changes and run `npm run lint`, `npm run typecheck`, and `npm test`.
4. Ensure all tests and linter pass before submitting.
5. Create a descriptive Pull Request using our standard template.

## Code Style
- We follow strict TypeScript and Next.js conventions.
- Run `npm run lint` to auto-fix styling issues.
- Pre-commit hooks (`.pre-commit-config.yaml`) will ensure formatting on every commit.

## Reporting Issues
- Use GitHub "Issues" to report bugs or suggest features.
- Provide steps to reproduce, expected behavior, and any relevant logs or screenshots.

## Additional Notes
Please see our [Code of Conduct](CODE_OF_CONDUCT.md) to understand community guidelines.
